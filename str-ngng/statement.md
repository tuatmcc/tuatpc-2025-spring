# `ngng` 文字列

## 問題文

英子文字からなる長さ $N$ の文字列 $S$ が与えられます。$S$ の $i$ 文字目を $S_i$ とします。以下の形式で表される $Q$ 個のクエリに答えてください。

- 整数 $l, r$ が与えられる。次の条件をすべて満たす整数の組 $(a, b, c, d)$ の数を答えよ。
    - $l \le a < b < c < d \le r$
    - $S_a = S_c$
    - $S_b = S_d$
    - $S_a \ne S_b$

-->

## 制約

- $4 \le N \le 3 \times 10^4$
- $1 \le Q \le 5 \times 10^4$
- $S$ は英子文字 `a-z` からなる文字列
- 各クエリについて、$1 \le l \le r \le N$
